## 機能

- TreeMage（GUIアプリ）
  - 系統樹の読み込み・表示
  - 簡単な樹形操作
  - 色付けやラベル付けといった出版物に利用するような系統樹の図の作成
- TreeMage.Core（.NET用ライブラリ）
  - 系統樹の読み込み
  - 樹形操作
  - 画像ファイル・PDFファイルへの出力・スタイリング
  - GUIアプリのプロジェクトファイル解析
